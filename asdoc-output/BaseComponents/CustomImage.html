public function CustomImage(scriptUrl:String, relPath:String, navigate:Boolean = false)

Constructor. Requires an url and a relative path to the image file. Will automatically load the image as a base64 string and decode it to an image source.

Parameters
scriptUrl:String — the url to the script which loads the image
 
relPath:String — the path parameter passed to the url via POST
 
navigate:Boolean (default = false) — for debugging purposes, navigates to the url of the script
