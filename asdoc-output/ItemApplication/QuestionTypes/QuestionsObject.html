Classpublic class QuestionsObject
InheritanceQuestionsObject CustomEventDispatcher flash.events.EventDispatcher
Implements ICustomEventDispatcher
Subclasses Cloze, ClozeNChoice, ItemDummy, MultipleChoice, SelectAndWrite, SingleChoice

Defines the basic structure and bahvior of a question object. The detailed behavior like load and display data is defined in each of the extending quesion classes.

A special feature is the load content by page behavior. All Data will be loaded immediately after calling the specific questionspage and will be unloaded by calling another questionpage. This frees memory and enables a fluid application.

protected function nextQpage(event:MouseEvent):void

This Method loops the reference array, searching for the Radiobutton Reference and get its ID and selection status. These information represents the user input and is fundamental for evaluation. Evaluation is always done in ItemObject. The clear screen behavior is not defined here but rather the change of the qpState.
