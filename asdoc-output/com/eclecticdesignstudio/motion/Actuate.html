update()method 
public static function update(target:Function, duration:Number, start:Array = null, end:Array = null, overwrite:Boolean = true):GenericActuator

Creates a new tween that updates a method rather than setting the properties of an object

Parameters

target:Function — The method to update
 
duration:Number — The length of the tween in seconds
 
start:Array (default = null) — The starting parameters of the method call. You may use both numeric and non-numeric values
 
end:Array (default = null) — The ending parameters of the method call. You may use both numeric and non-numeric values, but the signature should match the start parameters
 
overwrite:Boolean (default = true) — Sets whether previous tweens for the same target and properties will be overwritten (Default is true)

Returns
GenericActuator — The current actuator instance, which can be used to apply properties like ease, delay, onComplete or onUpdate

Example
Actuate.update (trace, 1, ["Value: ", 0], ["", 1]).onComplete (trace, "Finished tracing values between 0 and 1");
