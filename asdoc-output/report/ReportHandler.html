public function callReport(userid:String, thresholdvalue:int, dim:String, count:int, navigate:Boolean = false):void

Calls the user report from the server.

use the following PHP VARIABLES: $user = $_POST['user']; $thresh = $_POST['threshold']; $dim = $_POST['dimension']; $count = $_POST['listCount']; $script = $_POST['scriptCall'];

Parameters

userid:String — the id of the user
 
thresholdvalue:int — DEPRECATED the threshold of when a collection is solved correct
 
dim:String — DEPRECATED the dimension to evaluate
 
count:int — DEPRECATED indicates how many results per section will be displayed
 
navigate:Boolean (default = false) — allows to navigate to the php script if desired to debug php output
