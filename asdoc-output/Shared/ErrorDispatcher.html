public static function processNewError(error:String, description:String, flashErrorId:String, causedBy:String, line:String, scriptUrl:String, navigate:Boolean = false):void

creates an sends a new error message by combining the parameters and adding a timestamp. Then a urlloader will be created to request the specific php script which will write the message into a log file.

Parameters

error:String — custom number of error type
 
description:String — custom description of error type
 
flashErrorId:String — original falsh error id
 
causedBy:String — the source of the error
 
line:String — the module and if important also the line (line is usually useless, due to continual development)
 
scriptUrl:String — the url to the php script
 
navigate:Boolean (default = false) — for debugging, shows the target url
