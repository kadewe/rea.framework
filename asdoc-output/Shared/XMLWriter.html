public function XMLWriter(_relUrl:String, _xml:XML, _date:String, _name:String, _user:String = test, _param:Number = 0)

Constructor with presettings of parameters. The parameters representing the values which are required to make PHP write a new XML document with the correct information.

Parameters
_relUrl:String — The complete item XML File (required, cannot be null)
 
_xml:XML — The timestamp, given by the itemObject
 
_date:String — the complete Document name to be written
 
_name:String — the user name / iD (default ID is test)
 
_user:String (default = test) — there are different types of php scripts, whhich can be requested, the parameter indicates which specific script will be called.
 
_param:Number (default = 0)
