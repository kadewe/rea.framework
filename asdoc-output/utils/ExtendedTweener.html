public static function tween(target:Object, duration:Number, properties:Object, onCompleteFunc:Function = null, delay:Number = 0):void

Tweens a target's properties (if existent) in a given time towards a given value. Startvalues are target's current values.

Parameters

target:Object — The target to be tweened
 
duration:Number — The duration in seconds. If a half second is desired, then this value would be 0.5
 
properties:Object — An arbitrary object containing properties, which are intended to be tweened on the target. Note, that property names have to be the exact name (watch cases!) of the target's property.
 
onCompleteFunc:Function (default = null) — A function to be called, once the tween has been completed.
 
delay:Number (default = 0) — optional predelay in seconds.
