getRandomSquarePosition()method
public static function getRandomSquarePosition(width:Number, height:Number, boundx:Number, boundy:Number, compare:Array, maxIndex:uint):Array

This is a full working shuffle function. It auto-increases the y-bounds, if there is no more possible placement of an element. The max effort is n, before the bounds will increase.

Parameters

width:Number — the width of the current element, which will be compared with the others
 
height:Number — the height of the current element, which will be compared with the others
 
boundx:Number — the maxwidth of the elements target container
 
boundy:Number — the maxheight of the elements targetcontainer
 
compare:Array — the array of all elements to compare with
 
maxIndex:uint — the maximum amount of steps in the compare loop. assume there are only a few elements with a position set before, so the loop breaks at those who still have not obtained a position.

Returns
Array
